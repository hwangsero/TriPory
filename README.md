# TriPory

## 프로젝트 요약
### 개발기간
2018/12/26 ~ 2019/01/17

### 참여인원
4명

### 개요
- 여행을 다녀와서 찍은 사진들을 여행일지와 함께 정리하고 다른 이용자들과 공유할 수 있는 사이트

### 주요기능
- 여러 개의 사진을 업로드하면 일차별로 사진들을 자동 분류
- 업로드된 사진의 위치를 지도에 마커로 표시
- 작성된 글 목록을 무한스크롤 형식으로 보여줌
- 여행일지 좋아요, 스크랩
